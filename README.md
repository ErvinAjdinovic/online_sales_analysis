
## Features
- Create and manage products with name, price, and quantity
- Update product quantities
- Display product information
- Manage a collection of products using ProductManager
- Add and remove products from the system
- Manage a shopping cart and calculate total purchase value
- Version control using Git branches, commits, and merges

## Project Structure
- product.py – defines the Product class
- product_manager.py – manages product inventory
- cart.py – manages shopping cart functionality
- main.py – runs the application and demonstrates functionality

## Technologies Used
- Python 3
- Git
- GitHub
